# Conventions to problems

<center>

| Convention    | Description |
| -------- | ------- |
| SOLVED  | I was able to solve the problem    |
| SOLVED_WITH_HELP | I couldn't figured out how to solve, so i search for help     |
| IN_PROGRESS    | I could think about a solution but the algorithm isn't correct yet    |
| FAILED    | I tried but i couldn't make anything work   |
| NO_IDEA    | Need to study about the subject because i have no idea how to solve the question   |
